# ComPewter Acceptance Checklist

- [ ] New save: open ComPewter with `F8`.
- [ ] Existing save: open ComPewter with `F8`.
- [ ] Provider disabled: submit a question and see setup guidance.
- [ ] Ollama configured: ask "What should I plant today?" and receive an answer.
- [ ] OpenAI invalid key: see a friendly auth error without a crash.
- [ ] Anthropic invalid key: see a friendly auth error without a crash.
- [ ] Ollama stopped: see an unavailable-provider error without a crash.
- [ ] Context off: prompt works without sending game context.
- [ ] Context on: daily planning answer changes based on season/date/weather.
- [ ] Spoilers off: answer avoids unsolicited spoiler-heavy details.
- [ ] Remove mod: save remains playable.
- [ ] Multiplayer smoke: chat opens locally and does not affect other players.
